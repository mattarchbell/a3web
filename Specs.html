Setting Up For Success

Wifi

Here are the recommended Wifi requirements to get you up and running and to make sure you are able to attend virtual classes also share all your hard work:

  • 1. Upload Speed
  • Recommended upload internet speed is: 10 mbps or greater.
  • 2: Download Speed
  • Recommended download speed is: 1 mbps or greater.
  • 3. Ping and Latency: 40-50ms or less.
  • How do you find out if your internet is up to speed? You can run a speed test that will help you find out all of these things. If you are stuck you can use this speed test.

Specs

Another thing to consider is computer specifications. This is a big requirement as a lot of the software that will be used require certain specs to be able to run smoothly.

  • Here are the recommendations:
  • 1: Ram: 16 gb ram.
  • 2: Processor: i5 or i7, Windows 10.
  • 3: Storage: The more the better!
  • 4: Graphics: 64 MB RAM minimum; 256 MB RAM or higher recommended.

SetUp

For the ultimate experience and ease of learning GIS skills, a good set up will get you a long way.

  • Here are the tools recommended for a great set up:
  • 1: Desk Setup: It is super important to have a good workspace.
  • 2: A Dual monitor set up is helpful when using programs and it also helps save time!
  • 2: External hard-drive: This is a must for backing up data! GIS is data heavy and can require a lot of storage.

It is good to have back ups and extra storage in case your system is running low. Don't want all that hard work going and hours going to waste.
